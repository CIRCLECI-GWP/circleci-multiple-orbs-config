# Cakes CLI  application

Node JS Command line application with Commander.js

## Install Dependencies
```bash
npm install
```

## Run the application
```bash
node ./bin/index.js
```

## Order Cake
```bash
node ./bin/index.js order-cake
```

## List Cakes
```bash
node ./bin/index.js list
```

## Run tests
```bash
npm test
```
